# Soporte — Estadística Inferencial (marimo)

Este es un proyecto educativo **open-source y sin financiación**. No hay SLA, soporte comercial ni equipo dedicado.

## Dónde buscar ayuda

| Tipo de pregunta | Dónde |
|------------------|-------|
| **Bug / error en el notebook** | [Issues](https://github.com/tu-usuario/estadisticca/issues) → plantilla *Bug report* |
| **Petición de funcionalidad** | [Issues](https://github.com/tu-usuario/estadisticca/issues) → plantilla *Feature request* |
| **Duda pedagógica / estadística** | Abre un *Discussion* (pestaña Discussions) → etiqueta *Q&A* |
| **Problema de seguridad real** | Ver [SECURITY.md](SECURITY.md) — **no abras issue público** |

## Antes de abrir un issue

1. Ejecuta `marimo run main.py` y confirma que el error.
2. Busca en [issues existentes](https://github.com/tu-usuario/estadisticca/issues) (abiertos y cerrados).
3. Incluye:
   - Python version (`python --version`)
   - `uv sync` / `pip list` relevante
   - Pasos mínimos para reproducir
   - Captura si es visual

## Comunidad

- **Discussions**: preguntas, ideas, compartir uso en clase, traducciones.
- **PRs bienvenidos**: ver [CONTRIBUTING.md](CONTRIBUTING.md).

## Qué NO es soporte

- Clases particulares / tutoría estadística.
- Adaptación a currículo específico (¡haz un fork!).
- Debug de tu entorno local (venv, drivers GPU, etc.).
